# GKX API (NestJS)

Backend principal del proyecto GKX.

## Ejecutar en desarrollo

Desde la raíz del monorepo:

```bash
npm run dev:api
```

Desde esta carpeta:

```bash
npm run start:dev
```

## Build

Desde la raíz:

```bash
npm run build:api
```

Desde esta carpeta:

```bash
npm run build
```

## Tests y lint

```bash
npm run test
npm run test:e2e
npm run lint
```

## Variables de entorno

Usar `.env` local basado en `.env.example`.
