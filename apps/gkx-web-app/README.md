# GKX Web App (Next.js)

Frontend del proyecto GKX construido con Next.js (App Router).

## Ejecutar en desarrollo

Desde la raíz del monorepo:

```bash
npm run dev:web
```

Desde esta carpeta:

```bash
npm run dev
```

## Build

Desde la raíz:

```bash
npm run build:web
```

Desde esta carpeta:

```bash
npm run build
```

## Tests y lint

```bash
npm run test
npm run lint
```

## Variables de entorno

Usar `.env` local basado en `.env.example`.

## Contrato con backend

El contrato OpenAPI consumido por frontend está en `openapi.json`.
